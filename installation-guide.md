# Installation Guide

## Overview

Rosetta DBT Studio is available for Mac, Windows, and Linux. 
No prior Python or dbt™ installation is required — everything 
is bundled inside the application.

---

## Download & Installation

1. Navigate to the [Releases Page](https://github.com/rosettadb/dbt-studio/releases) on GitHub
2. Select the latest release
3. Download the correct file for your operating system:
   - **Mac (Apple Silicon M1/M2/M3):** `arm64.dmg`
   - **Mac (Intel):** `x64.dmg`
   - **Windows:** `.exe`
   - **Linux:** `.deb`
4. Open the downloaded file and follow the standard 
   installation steps for your operating system

> **Note:** Ignore the `.yml` and `.blockmap` files on 
> the releases page, those are for the app's internal 
> auto-updater.
